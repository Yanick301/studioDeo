# **App Name**: EZCENTIALS

## Core Features:

- Category Display: Display categories (Men's Clothing, Women's Clothing, Accessories, Winter Clothing) in a horizontal scroll menu with images, linking to respective category pages.
- Product Listings: Display products with image, name, price, and 'Add to Cart' button on category pages. Allow filtering and sorting by price, popularity, and latest.
- Product Details: Display detailed product information, including a product gallery, name, description, price, and options/sizes. Show at least 5 placeholder reviews.
- Shopping Cart: Implement a fully functional shopping cart allowing users to add, remove, edit items, view subtotal, total, and taxes.
- Secure Checkout: Integrate Stripe for secure payment processing, requiring all fields before payment. Send automatic confirmation emails after purchase. Send automatic confirmation emails after purchase.
- User Authentication: Enable user sign-up and login with Google OAuth, email verification, and reminder emails.
- Translation Tool: The website supports multiple languages and translates texts accordingly

## Style Guidelines:

- Primary color: The grey from the logo to evoke a sense of sophistication.
- Secondary color: The salmon color from the logo, providing a soft, elegant backdrop.
- Background color: white to highlight key elements and calls to action.
- Headline font: A modern sans-serif similar to the Logo Font, geometric, high contrast thin-thick lines, with an elegant, fashionable, high-end feel.
- Body font: PT Sans to pair with the headline font above.
- Use minimalist line icons for categories and navigation to maintain a clean and elegant aesthetic.
- Implement a mobile-first, fully responsive layout with smooth scrolling, hover effects, and elegant animations. Prioritize clear, uncluttered design to emphasize luxury.
- Subtle fade-in animations for content loading and smooth transitions between pages for enhanced user experience.
